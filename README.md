## Installation

To run PhotoFolio locally, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/Ananthuak16/PhotoFolio.git
cd photofolio
```

2. Install dependencies:

```bash
npm install
```

3. Start the application:

```bash
npm start
```

Visit [http://localhost:3000](http://localhost:3000) in your browser to use PhotoFolio.

## Usage

Once the application is running, you can:

- Create a new album by clicking on "Create a New Album."
- Add images to an album by clicking on the album and using the "Add Image" button.
- View the list of albums and click on them to see their images.
